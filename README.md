## Веб-прототип для распознавания объектов на изображениях с использованием предобученной модели машинного обучения.

Demo: https://epheemeeral.github.io/image-classifier/

# О проекте
Проект демонстрирует возможность быстрого внедрения AI-функционала в пользовательский интерфейс без разработки серверной части.
Пользователь загружает изображение, после чего приложение:
- анализирует его с помощью нейросети
- определяет наиболее вероятные объекты
- выводит результаты с вероятностями
Вся обработка выполняется в браузере (client-side).

# Как использовать
1) Нажмите «Выбрать файл»
2) Загрузите изображение
3) Нажмите кнопку распознавания
4) Дождитесь результатов классификации
### Время загрузки при первом запуске может быть больше, так как модель загружается в браузер

# Технологии
- JavaScript
- TensorFlow.js
- MobileNet (предобученная модель)
- HTML / CSS
